**网站：** [hxpcs.org](https://www.hxpcs.org/)

### 基本信息

| 项目      | 详情                                                                                   |
| --------- | -------------------------------------------------------------------------------------- |
| 上课地点  | Mercer County Community College，1200 Old Trenton Rd, West Windsor Township, NJ 08550 |
| 上课时间  | 每周日 1:30 PM – 5:00 PM                                                               |
| 学年      | 9月至次年6月（周日上课，节假日除外）                                                   |
| 年级范围  | 学前班 – 9年级，另有高级中文班                                                         |
| 字体/拼音 | 简体中文，普通话，拼音                                                                 |
| 招生规模  | 约 800 名学生，华夏系统规模最大分校之一                                                |

### 课程设置

- **中文主课**：幼儿园至9年级系统中文课，另设高级中文
- **CSL（中文作为第二语言）**：适合家庭语言非中文的孩子，双语授课

- **舞蹈团**：兰云舞蹈团（LanYun Dance）
- **足球俱乐部**

### 夏令营

每年暑假约10周（6月底至8月底），地点在 99 Clarksville Rd, West Windsor。内容包括：

- 学科：英语、数学、中文
- 体育：足球、篮球、乒乓球、击剑、游泳（另收费）
- 活动：乐高、编程、STEM、艺术、合唱、演讲

兄弟姐妹有折扣，报名通过 Google Form。

### 年度活动

春节晚会、运动会、书法/绘画比赛、朗诵比赛、毕业典礼

### 报名

- **注册入口**：[hxpcs.org/Registration](https://www.hxpcs.org/Registration/default.aspx)
- 新家庭需创建账号，老家庭用家庭电话查询原账号
- 学费信息登录后可见（未在官网公开）
- 联系方式：646-961-7900（工作日/周末 7PM 后）| registration@hxpcs.org

## 英华中文学校（YingHua Chinese School）

**网站：** [yinghua.org](https://www.yinghua.org/)

### 基本信息

| 项目      | 详情                                                    |
| --------- | ------------------------------------------------------- |
| 上课地点  | Chapin School，4101 Princeton Pike, Princeton, NJ 08540 |
| 上课时间  | 每周日两场：1:35–3:20 PM 或 3:30–5:20 PM                |
| 学年      | 约31周，9月至次年6月                                    |
| 年级范围  | 幼儿园 – 成人（Grade 10 AP班）                          |
| 字体/拼音 | 简体中文，普通话，拼音辅助                              |
| 教学理念  | 马立平直接认字法（Dr. Liping Ma 开发）                  |

### 课程轨道

英华有三条教学轨道，按家庭语言背景分班：

- **CHL（中文传承语言班）**：家中说中文的孩子，使用马立平教材，共11个级别
- **CSL（中文作为第二语言班）**：家中不说中文，使用 QTalk 教材，共6个级别
- **CBL（中文背景语言班）**：家庭中文环境有限，使用双双中文教材
- **Grade 10 AP 班**：高中生可修AP中文

英华是全美首批获得 CESS-MSA（中部各州协会）认证的社区中文学校。

| 家长义工押金（履行义工职责后退还） | $50      |
| **合计**                           | **$690** |
| AP班合计                           | $750     |
| 早鸟优惠（6月初截止）              | 减 $30   |

付款方式：Zelle 至 treasurer@yinghua.org，或支票抬头 "YingHua Language School"。

### 课外选修

网球、围棋（6-10岁）、瑜伽、英文写作、绘画、演讲（中文表达课，由校长宁辛老师亲授）

### 暑期强化班

马立平暑期中文强化班（约10周），有早鸟和兄弟姐妹折扣，报名通过 Google Form。

另设 **PCS 课后班**：在 Princeton Charter School 的工作日课后中文辅导（马立平班和CSL班）。

### 家长职责

英华实行家长义工制度：每个家庭需完成一定义工时长，$50 押金在履行义工后退还。学校要求家长积极在家配合——尤其是 CHL 班的马立平教学法，需要家长在家坚持说中文。

课业要求：每周约4份作业（每份20-30分钟），每学年3次单元考试（笔试+口试）。

### 报名

- **注册信息**：[yinghua.org/register](https://www.yinghua.org/register/registration-info)
- 每年1月中旬开放报名，先到先得，部分班级有名额限制
- 联系：admission@yinghua.org | admin@yinghua.org

## 选校建议

**选华夏普兰斯堡**：规模最大、活动最丰富，适合想让孩子有更多华人社交圈、同时参与课外活动（武术、围棋、舞蹈等）的家庭。

**选华夏蒙哥马利**：住在 Montgomery 或 Skillman 一带、或者周日不方便的家庭首选。周六上课，通勤距离近。

**选英华**：如果在家坚持说中文，对中文教学系统性有较高要求，英华的马立平教法口碑很好，适合把"真正学会中文"放在首位的家庭。学费信息公开透明也是加分项。
